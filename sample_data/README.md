# Sample Data

This folder contains anonymized sample files used to demonstrate the automation workflow.

## Input

- **UDM_Dispute.xlsx** – Sample customer dispute database.
- **UDM_Notes.xlsx** – Sample dispute notes used to enrich the master dataset.

## Output

Contains example workbooks generated by the automation, organized by region.

> **Note:** All data has been anonymized and does not contain any confidential or proprietary business information.
